# Proto campagne BIA

Prototype HTML statique — flow campagne BIA (GLI Individual Enrollment Certificate).

## Prévisualisation locale

```bash
npx --yes serve . -p 3456
```

Puis ouvrir : http://localhost:3456/prototype/Proto_campagne_BIA/

## GitHub Pages

1. Repo GitHub **Proto_campagne_BIA** (vide, sans README auto).
2. Pousser ce dossier sur `main`.
3. Settings → Pages → Source : **Deploy from branch** → `main` / **/(root)**.
4. URL : `https://<username>.github.io/Proto_campagne_BIA/prototype/Proto_campagne_BIA/`

> Regénérer depuis le monorepo : `bash scripts/publish-proto-campagne-bia.sh` (à la racine de Design_prototype).
